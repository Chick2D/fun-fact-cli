# fun-fact-cli
---
Displays fun facts in your terminal, literally that's it

# Installation
---
1. Run
```bash
git clone https://github.com/Chick2D/fun-fact-cli/
chmod +x fun-fact-cli/funfacts.sh
```
Run the shell script using `./funfacts.sh`, that's it
